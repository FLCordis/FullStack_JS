Trabalhando com Array

Manipulando arrays com:

  • Push - Adicionando elemento no final do Array.
  • Unshift - Adicionando no começo do array.
  • Pop - Removendo elementos no final.
  • Shift - Removendo elementos no começo.
  • Includes - Pesquisa de elementos, retornando TRUE ou FALSE;
  • IndexOf - Descobrindo a posição que o valor está no array.
  • Slice - Corta posições especificas entre A e B e concatena depois.
    (O valor negativo inverte a contagem, indo do final para o começo - ex: -1 é o último valor, -2 é o penultimo e etc).
  • Concat - Soma dois arrays.
  • Splice - Substitui apartir de um lugar os elementos e devolve em uma lista.
