Cadastro de Carros

  • Deve ter um menu interativo que sempre é exibido até que o usuário escolha sair.
  • O menu interativo deve mostrar no topo a quantidade de carros cadastrados.
  • O menu deve ter a opção de salvar um carro.
  • Para salvar um novo carro o programa deve pedir as seguintes informações:
    • Nome
    • Fabricante
    • Modelo
    • Ano
    • Cor
  • O menu também deve ter a opção de mostrar todos os carros salvos.
